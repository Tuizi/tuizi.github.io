So between using a border or using an icon to render an information there is not perseptible impact on performance for the user. The performance impact is more precisly in font used for the icons, if you need to download one only to display these icons so border will be better, but most of the time a webfont is already there to display all icons in your page, in this case border or icons have the same impact on rendering. Keep in mind, often we add a i tag to add these icon instead of using a ::before, this will increase the parseHTML time.
